# Portfolio case study

This is a dependency-free static portfolio page for the G1 XR Teleoperation project.

From the repository root, preview it with:

```powershell
python -m http.server 8080
```

Then open `http://localhost:8080/portfolio/`.

The page references the edited Quest demo video in `captures/quest3/2026-07-13/` and can be hosted from the repository root with GitHub Pages or another static host.
